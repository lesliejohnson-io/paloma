## AI coaching companion for the space between sessions.

Paloma helps coaching clients reflect, regulate, and sustain momentum 
between live sessions. It surfaces a client's own evidence against their 
limiting beliefs — and prepares both client and coach for what's next.

---

## What It Does

- Logs small acts of courage and tags them to a limiting belief
- Builds an evidence file that compounds over time
- Detects patterns and surfaces them at the right moment
- Generates session prep summaries for client and coach
- Maintains a 5-layer memory architecture calibrated for trust

---

## Memory Layers

| Layer | What It Stores | Persistence |
|---|---|---|
| L1 Working Memory | Conversational context | Resets each session |
| L2 Episodic Summary | Session takeaways | User-approved only |
| L3 Semantic Identity | Strengths, values, identity | Human-reviewed |
| L4 Action & Progress | Commitments, wins | Longitudinal |
| L5 Regulation Profile | Grounding techniques | Always available |
